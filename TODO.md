# 待优化清单

> 仅供参考，需用户明确指令后才执行。

## 性能优化

- [ ] **自适应降采样** — 检测帧率低于阈值时自动缩小处理分辨率，处理完放大回原尺寸
- [ ] **隔帧处理** — 每 N 帧处理 1 帧，降低 CPU 负载，视觉差异小
- [ ] **OffscreenCanvas** — 将 Canvas 绑定到 Worker 线程，省掉主线程 putImageData 开销
- [ ] **SIMD 加速** — Rust 启用 WASM SIMD 指令，像素批量运算并行化
- [ ] **多 Worker 并行** — 将图像分块交给多个 Worker 同时处理

## 功能扩展

- [ ] **更多视频特效** — 边缘检测（Sobel）、锐化、色彩空间转换、灰度等
- [ ] **特效叠加** — 支持多个特效串联（如先模糊再红外）
- [ ] **录制导出** — 将处理后的视频流录制为 WebM 下载
- [ ] **摄像头分辨率选择** — 支持切换 720p/1080p 等分辨率
